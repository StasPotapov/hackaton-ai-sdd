## Шаг 4. Итерации уточнений

**Метод:** для каждой из 893 задач — анализ комментариев до даты первого перехода в `In Progress`. Поиск вопросительных знаков и сигнальных слов (уточните, непонятно, какой/какие, нужен макет, критерии приёмки и т.п.).

**Результат (автоматический анализ, все 893 задачи за H1 2026):**

| Параметр | Значение |
|---|---|
| **Среднее число уточняющих итераций** | **≈0.118** |
| Всего уточняющих вопросов | 105 на 893 задачи |
| Задач с хотя бы одним уточнением | 62 из 893 (6.9%) |
